# gba-aseprite-shader

GBA Aseprite Shader

## Goal

Recreate the original Game Boy Advance screen look in digital pixelart.

## Effects

- low contrast ratio
- saturation
- blue-green tint
- red: orange / brown
- green: less vibrant
- blue: cyan / teal
- black appears as dark grey
- magenta/purple shift in darker tones
